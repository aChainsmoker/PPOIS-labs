log - обновляет отображение состояния машины после каждого действия
exe [true/false] - выполняет записанные инструкции (пошагово, если true)
edit [instruction] - редактирует иструкцию 
delete [instruction number] - удаляет инструкцию
add [instruction] - добавляет инструкцию
length [n] - меняеть область отображения ленты 
mark - ставит метку на ленте
erase - убирает метку с ленты
right - перемещает картку на клетку правее
left - перемещает картку на клетку левее
switch - меняет режим выполнения инструкций (только во время выполнения)
next - следующий шаг выполнения программы
exit - выход
